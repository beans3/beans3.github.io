### 2. Boolean
- Python에서 True와 False의 대소문자 구분을 주의해야 한다.  
(TRUE, tRue, FALSE, fAlse 모두 변수로 인식한다.)
- True와 False의 값  
```python
True == 1 # True
False == 0 # True
```  

- 1과 "1"은 다르다  
```python
True == "1" # False
False == "0" # False
```  

- 논리식으로 True/False 판단  
```python
True and False # True
True or Fasle # True
```

> 'Python Event 2019'

```python
p.upper()
```  

'python enevnt 2019'

### 4. for문
- for문 규칙  
- 콜론(:) 다음 탭이 되지 않으면 오류 발생 (Indentation 오류)  
탭이 괄호 역할을 하기 때문  
- 보통 탭 한 번 = 띄어쓰기 네 번  
```python
for a in p_list:
    print(a)
```  
> 서울시  
> 강남구  
> 삼성동  
> 100-1  

- for문 안에 if문  
```python
for a in p_list:
    if "서울" in a:
        city = a
        print("조건문 : 서울")
    else:
        print("else 안 :", a)
        #print("else 안 :", + a)도 가능
    print("반복문 안")
print("---반복문 끝---")
print(city)
```  
> 조건문 : 서울  
> 반복문 안  
> else 안 : 강남구  
> 반복문 안  
> else 안 : 삼성동  
> 반복문 안  
> else 안 : 코엑스  
> 반복문 안  
> else 안 : 203호  
> 반복문 안  
> ---반복문 끝---  
> 서울시  

### 5. Dictionaries (dict 자료형)
- dict 자료형(key:value)  
```python
weekday_map = {0:'월', 1:'화', 2:'수', 3:'목',4:'금', 5:'토', 6:'일'}
weekday_map
```  
> {0: '월', 1: '화', 2: '수', 3: '목', 4: '금', 5: '토', 6: '일'}  

- 리스트가 아니고 dict 자료형(key:value)으로 인식  
```python
type(weekday_map)
```  
> dict  

- key 값으로 value 출력하기  
```python
weekday_map[0]
```  
> '월'  
> 리스트 값이 아니라 키 값 '월'을 출력한 것  
> 만약 weekday_map = {1004:'월', 1:'화', 2:'수', 3:'목',4:'금', 5:'토', 6:'일'}라면  
> weekday_map[1004]의 출력값은 '월'  

- dic 자료형의 길이  
```python
len(weekday_map)
```  
> 7  

- for문으로 key 값 출력하기  
```python
for day in weekday_map:
    print(day)
```  
> 0  
> 1  
> 2  
> 3  
> 4  
> 5  
> 6  

- for문으로 value 값 출력하기  
```python
for day in weekday_map:
    print(weekday_map[day])
```  
> 월  
> 화  
> 수  
> 목  
> 금  
> 토  
> 일  

### 6. 함수 정의
- 주소를 반환하는 함수 만들기  
```python
def extract_p(p):
    return p
extract_p(p)
```  
> ' 서울시 강남구 &nbsp;삼성동 100-1 '  

- 주소에서 '시'를 반환하는 함수 만들기  
```python
def extract_city(p):
    city = p.split()[0]
    return city
extract_city(p)
```  
> '서울시'  

- 주소에서 '구'를 반환하는 함수 만들기  
```python
def extract_gu(p):
    gu = p.split()[1]
    return gu
extract_gu(p)
```  
> '강남구'  

### 7. 리스트의 append와 extend
- append로 리스트에 자료를 추가해봅시다.  
```python
gu = ["서대문구", "마포구"]
gu.append("용산구")
gu
```  
> ['서대문구', '마포구', '용산구']  

- pop()으로 요소 삭제하기 : 맨 마지막 요소를 삭제  
```python
gu.pop()
gu
```  
>  '용산구'  
> ['서대문구', '마포구']  

- append로 리스트 자료를 추가해봅시다.  
```python
gu = ["서대문구", "마포구"]
gu.append(p_list)
gu
```  
> ['서대문구', '마포구', ['서울시', '강남구', '삼성동', '100-1']]  
> 리스트 전체를 하나의 원소로 인식한다는 것을 알 수 있음  

- extend로 리스트에 자료를 추가해봅시다.  
```python
gu.extend(p_list)
gu
```  
> ['서대문구',  
> '마포구',  
> ['서울시', '강남구', '삼성동', '100-1'],  
> '서울시',  
> '강남구',  
> '삼성동',  
> '100-1']  
 
 - remove()로 특정 요소 제거하기  
```python
gu.remove("서울시")
gu
```  
> ['서대문구',  
> '마포구',  
> ['서울시', '강남구', '삼성동', '100-1'],  
> '강남구',  
> '삼성동',  
> '100-1']
